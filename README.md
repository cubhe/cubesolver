## 11.7更新
目前正在写视觉部分

开发环境是win10＋VS2017   opencv3.4 is also required

逻辑大概是这样：

1. 调用两个摄像头分别读取三个面
2. 图像预处理 直方图均衡化 模糊
3. 手动分割并提取RGBHSV六个值的平均值 这样三个面会有 27*6个值
4. 调用opencv ml库里面的SVM 预测值并返回（需要提前训练好模型）
